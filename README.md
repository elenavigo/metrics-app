## Application

Metrics application! This app enables you to visualize and update metrics data.

## Repository

This repository includes:
- Frontend application using ReactJS
- Backend application using NestJS
- Docker configuration

## Requirements

Node.js 16+

## Project setup

```bash
$ npm install
```

## Database

```bash
$ docker-compose up -d
```

## Backend

```bash
$ npm run start:dev
```

## Frontend

```bash
$ npm start
```
